Limboole on the Go!

Uses Limboole and Lingeling to parse an easy SAT DSL (instead of relying on DIMACS). Compiled using Emscripten, Source Code and Modifications are available on GitHub. Syntax is improved for mobile devices: The / symbol also counts as OR (same as |). Every time a formula is executed, the link in your URL-bar (awesome-bar, location-bar or some other name) updates and can be copied. This link can be opened at other browsers and directly executes the given formula. Give it a try!

Created by Max Heisinger. I also wrote a short blog entry about this.
